===============================
Bottle-cuturl
===============================

A simple URL shortening app. Based on cookiecutter-bottle template.

Quickstart
----------

Development mode:

.. code-block:: bash

    init-dev.sh
    vex --path env python manage.py runserver

Test / Production (yay) mode:

.. code-block:: bash

    init-tst.sh
    vex --path env python manage.py runserver
